01 · TIPOGRAFIA

Três fontes, três vozes

O Recorte usa um sistema híbrido: uma sans-serif de display pra impactar, uma sans humanista pra ler confortável, uma serif italic pra acentuar com humanidade.

📡 Como carregar as fontes

Todas as 3 fontes vêm do Fontshare (Indian Type Foundry), CDN gratuito. Adicione em todos os HTMLs:

<!-- No <head> de cada HTML --> <link rel="preconnect" href="https://api.fontshare.com"> <link rel="stylesheet" href="https://api.fontshare.com/v2/css?f[]=switzer@400,500,700,800,900,400i,700i&f[]=general-sans@400,500,600,700&f[]=gambarino@400i,400&display=swap">

06 · VOICE

Como a gente fala

O Recorte é pra todo brasileiro — direto, próximo, sem encheção. Linguagem de conversa de café, não de relatório. Esses 7 princípios + a tabela "não use / use" guiam todo o conteúdo: edições, site, FAQ, microcopy, emails de transação. Quando estiver em dúvida sobre tom, volta aqui.

1

Direto, sem encheção

Frase curta. Verbo na frente. Se a frase pode ser cortada e ainda fazer sentido, corte.

2

Em segunda pessoa

Usa "você", "a gente". Não fala "o leitor", "o consumidor", "nós" formal.

3

Leve, não infantil

Max 1-2 emojis temáticos por manchete. Sem "rs", "kkk", caps lock. Respeito ao leitor.

4

Brasileiro real

Sem "bottom line", "spoiler", "heads up". Vira "resumindo", "adianta", "fica esperto".

5

Curioso, não cínico

"Repare:", "olha só:", "tem um detalhe:". Sem ironia gratuita, sem desdém.

6

Curadoria humana

"A gente recortou", "escolhemos pra você". Nunca "algoritmo", "IA", "filtro".

7

Encerra com gentileza

"Amanhã tem mais. Bom dia." Nunca "Continue acompanhando".

Tabela de substituição

Sempre que ver à esquerda, substitua pela versão à direita. Vale pro Claude (no system prompt) e pra qualquer humano escrevendo.

08 · IMPLEMENTAÇÃO

CSS base

Copie esse CSS pra todos os HTMLs do projeto. Define variáveis, fonte base, e os pesos disponíveis.

/* ============================== */ /* TIPOGRAFIA RECORTE — sistema híbrido */ /* ============================== */ /* CARREGAR FONTES (no <head>) */ <link rel="preconnect" href="https://api.fontshare.com"> <link rel="stylesheet" href="https://api.fontshare.com/v2/css?f[]=switzer@400,500,700,800,900,400i,700i&f[]=general-sans@400,500,600,700&f[]=gambarino@400i,400&display=swap"> /* CSS VARIABLES */ :root { /* Fontes */ --font-display: 'Switzer', -apple-system, BlinkMacSystemFont, sans-serif; --font-body: 'General Sans', -apple-system, BlinkMacSystemFont, sans-serif; --font-italic: 'Gambarino', Georgia, serif; --font-mono: 'JetBrains Mono', ui-monospace, monospace; } /* BASE */ body { font-family: var(--font-body); font-weight: 400; line-height: 1.65; color: var(--ink-soft); } /* HEADINGS */ h1, .H1 { font-family: var(--font-display); font-weight: 900; letter-spacing: -0.045em; line-height: 0.94; } h2, .H2 { font-family: var(--font-display); font-weight: 900; letter-spacing: -0.03em; line-height: 1.04; } h3, .H3 { font-family: var(--font-body); font-weight: 700; letter-spacing: -0.015em; line-height: 1.28; } /* ITALIC editorial (em destaques) */ .italic, em { font-family: var(--font-italic); font-style: italic; font-weight: 400; } /* Brand mention em texto */ .brand { font-family: var(--font-body); font-weight: 700; color: var(--ink); } /* Logo / wordmark */ .logo { font-family: var(--font-display); font-weight: 900; letter-spacing: -0.035em; color: var(--ink); } .logo .scissor { color: var(--mint-deep); } /* CTA button */ .cta { font-family: var(--font-display); font-weight: 800; font-size: 13px; letter-spacing: 0.12em; text-transform: uppercase; } /* Microcopy / kicker */ .micro { font-family: var(--font-display); font-weight: 800; font-size: 11px; letter-spacing: 0.22em; text-transform: uppercase; color: var(--ink-muted); } /* Stats / números */ .stat { font-family: var(--font-display); font-weight: 900; font-size: 60px; letter-spacing: -0.035em; color: var(--mint-deep); line-height: 1; }

Email template (HTML inline styles)

No email, as mesmas regras se aplicam mas com inline styles obrigatórios (Gmail/Outlook não suportam <style> em todos os contextos). Use:

/* Email — constantes a usar no Python */ SERIF_FONT_FALLBACK = "'Gambarino', Georgia, 'Times New Roman', serif" SANS_DISPLAY = "'Switzer', -apple-system, BlinkMacSystemFont, sans-serif" SANS_BODY = "'General Sans', -apple-system, BlinkMacSystemFont, sans-serif" /* Exemplo: manchete tema */ <h3 style="font-family:{SERIF_FONT_FALLBACK};font-weight:400;font-size:26px;line-height:1.22;letter-spacing:-0.01em;color:#1A1A1A;margin:0 0 12px;"> Anthropic anuncia Claude 5 </h3> /* Exemplo: resumo */ <p style="font-family:{SANS_BODY};font-weight:400;font-size:16px;line-height:1.65;color:#4A4A4A;margin:0;"> A empresa lançou hoje seu modelo mais capaz. </p>

Fallback chain

Caso o Fontshare CDN não carregue, a chain garante leitura aceitável:

Switzer → -apple-system → BlinkMacSystemFont → sans-serif General Sans → -apple-system → BlinkMacSystemFont → sans-serif Gambarino → Georgia → "Times New Roman" → serif
